## Gmail Assistant

A side panel extension for [Gmail](https://mail.google.com) that provides quick label-based filtering and navigation. Browse your labels in a multi-column layout, narrow by time scope, and jump to filtered views with one click.

<!-- [Install from Chrome Web Store](https://chromewebstore.google.com/detail/gmail-assistant/TODO) -->

### Features

- Browse all your Gmail labels in a multi-column layout
- Click a label to filter the Gmail page by that label
- Include sub-labels: selecting a parent label shows messages from all its children too
- System labels (Inbox, Sent, Starred, Important) shown alongside user labels
- Filter by time scope (1 week to 5 years)
- Dynamic label filtering — only shows labels that appear on matching messages
- Background cache for instant subsequent filtering
- Auto-hide side panel when leaving Gmail
- Zoom controls with per-context persistence
- Configurable keyboard shortcut to toggle the side panel

### Usage

1. Navigate to [Gmail](https://mail.google.com) in a tab
2. Click the extension icon to open the side panel
3. Filter your Gmail view:
   - **Labels** — click to filter, click again to deselect
   - **Scope from** — limit results to a time range
4. Click the "?" icon for a built-in help page

**Requires:** Google account authorization (Gmail read-only access via OAuth2).
